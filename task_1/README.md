# Flutter Shopping App UI

A simple Flutter Shopping App UI that includes a Login Screen and a Home Screen with a search bar and a welcome card. This project demonstrates basic Flutter UI design, navigation, form validation, and state management.

## Features

- Login Screen
- Email and Password Input
- Show / Hide Password
- Remember Me Checkbox
- Form Validation using SnackBar
- Navigation to Home Screen
- Search Bar in AppBar
- Gradient UI Design
- Category Icons
- Responsive Layout

## Screens

### Login Screen
- Email field
- Password field
- Password visibility toggle
- Remember me checkbox
- Login button
- Social login buttons

### Home Screen
- AppBar with title and action icons
- Search bar
- Welcome card with greeting
- Category icon containers

## Getting Started

### Prerequisites

Make sure you have installed:

- Flutter SDK
- Android Studio or VS Code
- Android Emulator or Physical Device

Check Flutter installation:

flutter doctor

### Clone the Repository

git clone https://github.com/ahmedhamzarana/flutter-shopping-app.git

### Navigate to Project Folder

cd flutter-shopping-app

### Install Dependencies

flutter pub get

### Run the App

flutter run
